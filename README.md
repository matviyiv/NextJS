# TaskFlow

[![CI — Lint](https://github.com/matviyiv/NextJS/actions/workflows/ci.yml/badge.svg)](https://github.com/matviyiv/NextJS/actions/workflows/ci.yml)
[![Deploy to GitHub Pages](https://github.com/matviyiv/NextJS/actions/workflows/deploy.yml/badge.svg)](https://github.com/matviyiv/NextJS/actions/workflows/deploy.yml)

A PWA task management app built with Next.js, Redux Toolkit, and Tailwind CSS.

## Live Demo

**[Open TaskFlow](https://matviyiv.github.io/NextJS/)**

## Features

- Create, edit, and delete tasks with priorities, statuses, and due dates
- Multiple views: List, Board (Kanban), Group, and Tag
- Subtask tracking with progress bars
- Real-time search filtering
- Export/import tasks as JSON
- Dark mode
- Offline-ready PWA with service worker
- Data persisted locally via redux-persist
